30-second orientation

  • It is a local Obsidian plugin. No account, no sign-up, no network calls, no servers — nothing leaves your vault.
  • Governance lives in frontmatter. Owners, status, and review dates are written into standard YAML, so your notes stay portable plain Markdown.
  • Six commands. From the Command Palette (Ctrl/Cmd-P, type Ledger): new governed document, set status, set owner, mark reviewed, governance dashboard, and open the audit log.
  • A dashboard and status bar show total docs, how many are ungoverned, and review freshness (never / stale / fresh) at a glance.
  • An in-vault audit log (_ledger/audit.md) records every action as a plain Markdown table you own.
  • Free and open source (MIT). New here? Start with Getting started.
