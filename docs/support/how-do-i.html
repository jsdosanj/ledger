Help Center · How do I…?

"How do I…?" recipe index

Short answers to common tasks, each linking to the full steps. Skim for what you need.

Pages & content

  • …create a new page? Click + New doc, enter a title and a folder. → Doc tree
  • …start from a template? Click Templates, pick a number, name it. → Templates
  • …make a new space (folder)? Give a page a new folder name — the space is created automatically. → Spaces
  • …edit a page with live preview? Open it → Edit tab → type → Ctrl/Cmd+S. → Editor
  • …rename a page? Click the title at the top and type a new one. → Editor
  • …delete a page? Open it (as Admin/Editor) → Delete. → Roles
  • …add tags? Use the tags box in the editor (comma-separated) or frontmatter. → Tags
  • …link two pages? Type [[Page Title]] in the body. → Wiki-links
  • …see what links to a page? Check the backlinks in the meta bar, or the Links tab. → Links view

Finding & reviewing

  • …search everything? Type in the top-bar search box (title, path, tags, frontmatter, body). → Search
  • …see a page's version history? Open it → History tab. → History
  • …compare or restore an old version? History tab → View diff / Restore. → History

Governance & roles

  • …mark a doc reviewed? Open it → governance bar → Mark reviewed today (stamps reviewed_on). → Governance
  • …assign an owner? Add owner: Name to the page's frontmatter. → Governance
  • …find stale docs? The governance bar flags "needs review" (older than 90 days); the Obsidian plugin has a full governance report. → Governance
  • …approve a version? As Admin/Reviewer, open the page → Approve this version. → Governance
  • …see what each role can do? Use the 🔑 role picker to switch roles. → Roles

Sharing & portals

  • …share a space read-only with an auditor? As Admin, click Publish ↗ on the space header; send the downloaded HTML. → Portals

Obsidian & sync

  • …import my Obsidian vault? Click Import vault, paste a note or bundle. → Import
  • …keep a folder of .md in sync two-way? Click ⇄ Vault sync → connect the folder (Chromium browsers). → Vault sync
  • …sync automatically? In the vault sync panel, tick Auto-sync. → Auto-sync
  • …work entirely inside Obsidian? Install the Obsidian plugin and push/pull the org vault. → Plugin
  • …sync on Firefox/Safari? Use Import/Export round-trip or a Chromium browser. → Fallback

Account, cloud & AI

  • …sign in? Click Sign in → Sign in with Clerk. → Sign in
  • …sync my KB across devices? Sign in, Sync KB up on one device, Pull from cloud on another. → Org vault
  • …publish compliance evidence? Signed in, click Publish documentation evidence. → Evidence
  • …set up AI? Click AI, choose provider, paste your key, Save key. → AI setup
  • …draft / summarize / ask my docs with AI? Use the three flows in the AI panel. → AI flows

Data, backup & leaving

  • …export one page? Open it → Export .md. → Export
  • …export everything / back up? Click Export all for a portable bundle. → Export
  • …handle a flagged secret? Remove it and link a vault item (or confirm if it's a false positive). → Flagged secret
  • …store a credential safely? Use the credential reference template and link your password manager — never paste the secret. → No-secrets
  • …start fresh / reset? Export first, then clear the site's storage and reload. → Reset
  • …leave with no lock-in? Export to Markdown, open in Obsidian or commit to git. → Leaving

Running & hosting

  • …run it locally? Serve over HTTP: python3 -m http.server 8000. → Self-host
  • …deploy on my own infra / a subpath? Any static host; relative paths support subpaths. → Self-host
  • …fix "stuck on Loading"? You opened file:// — serve over HTTP instead. → Troubleshooting
